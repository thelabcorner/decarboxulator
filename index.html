Expression Evaluation

Decarboxulator™ uses a powerful custom expression evaluation engine similar to that of Excel formulas!

In any input field simply press the '=' key, then type in a 'PEMDAS' math expression like 2+2 or (4*7) + (56+92) and press enter. The result will be displayed in the active input field. Try it out below!

Another example use case of our engine is inline unit conversion, for example, try this expression: "=2.4 lb to gram".

Or even more complex expressions like "=(2.4 lb to grams) + (2 oz to grams) in kilograms".

For more advanced calculations, you can use the MathJS syntax.
